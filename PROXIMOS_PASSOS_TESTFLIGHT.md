## 📋 Próximos Passos

### 1️⃣ Aguardar Processamento (10-30 minutos)

O build precisa ser processado pela Apple. Aguarde alguns minutos.

### 2️⃣ Acessar App Store Connect

1. Acesse: https://appstoreconnect.apple.com
2. Faça login com sua conta Apple Developer
3. Clique em **"My Apps"**

### 3️⃣ Criar o App (se ainda não criou)

1. Clique no botão **"+"** (criar novo app)
2. Preencha:
   - **Platform**: iOS
   - **Name**: Task2026 (ou o nome desejado)
   - **Primary Language**: Português (Brasil)
   - **Bundle ID**: Selecione `com.josepeieira.task2026` (ou o que você configurou)
   - **SKU**: Um identificador único (ex: `task2026-001`)
3. Clique em **"Create"**

### 4️⃣ Adicionar Build ao TestFlight

1. No App Store Connect, vá na aba **"TestFlight"**
2. Aguarde o build aparecer na seção **"iOS Builds"** (pode levar 10-30 minutos)
3. Quando aparecer, você verá o build listado

### 5️⃣ Configurar Testadores Internos

1. Vá em **"Internal Testing"**
2. Clique em **"+"** para criar um grupo (se necessário)
3. Adicione o build que você fez upload
4. Os usuários com acesso de desenvolvedor aparecerão automaticamente

### 6️⃣ Configurar Testadores Externos (Opcional)

1. Vá em **"External Testing"**
2. Clique em **"+"** para criar um grupo
3. Dê um nome (ex: "Beta Testers")
4. Adicione o build
5. Preencha as informações de teste (obrigatório na primeira vez):
   - O que testar
   - Instruções de feedback
6. Adicione os emails dos testadores
7. Envie o convite

**⚠️ Importante:** A primeira versão externa precisa ser aprovada pela Apple (pode levar 24-48 horas)

### 7️⃣ Instalar no iPhone

#### Para Você (Desenvolvedor):
1. Instale o app **TestFlight** da App Store no seu iPhone
2. Abra o TestFlight
3. Faça login com sua Apple ID de desenvolvedor
4. O app aparecerá automaticamente
5. Toque em **"Install"**

#### Para Testadores Externos:
1. Receberão um email de convite
2. Devem instalar o app **TestFlight** da App Store
3. Abrir o link do convite no email
4. Aceitar o convite
5. O app aparecerá no TestFlight
6. Toque em **"Install"**
